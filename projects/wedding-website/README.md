## 1. 项目目标

为 Human 的婚礼创建一个精美的婚礼网站，承担两个核心功能：
1. **电子请帖** — 替代/补充纸质请帖，方便分享给宾客
2. **婚礼记录** — 婚礼前展示倒计时和信息，婚礼后承载照片和回忆

### 设计理念

这是一个**非常个人化**的项目，视觉审美比功能数量更重要。目标是让宾客打开网站时感到**温馨、精致、有仪式感**，而不是一个功能堆砌的 SaaS 产品。

## 2. 竞品研究与设计参考

### 主流婚礼网站平台分析

| 平台 | 优势 | 不足 |
|------|------|------|
| **Zola** | 50+ 精美模板，现代审美，强大的 registry 集成 | 模板定制受限，免费版有品牌水印 |
| **WithJoy** | 简洁现代，Smart RSVP，免费无水印 | 设计选项较少 |
| **The Knot** | 最大婚礼平台，vendor 生态完整 | 设计偏传统，双语支持弱 |
| **WedSites** | 原生多语言支持（宾客按语言自动切换） | 模板较少 |
| **Riley & Grey** | 最高端的设计审美，极简优雅 | 收费 ($35/月) |

### 设计趋势（2025-2026）

1. **极简优雅** — 大量留白，serif 字体，柔和配色（奶白、米色、淡金、雾霾蓝）
2. **全屏英雄图** — 首屏大图或视频背景，营造电影感
3. **手绘/水彩插画** — 增加个人化和艺术感
4. **滚动叙事** — 单页滚动讲述爱情故事，而非多页跳转
5. **动画细节** — 微妙的 fade-in、parallax 效果，增加精致感
6. **暗色 vs 亮色** — 亮色（白+金）经典优雅；暗色（深蓝/墨绿+金）现代时尚

### 我们的设计方向

**推荐风格：现代优雅（Modern Elegant）**
- 配色：**暖白底 + 淡金色点缀 + 深色文字**（经典永恒，适合中西合璧婚礼）
- 字体：英文用优雅 serif（如 Playfair Display），中文用宋体或楷体风格 Web Font
- 布局：单页滚动式，各 section 自然过渡
- 图片：大幅婚纱照/情侣照作为视觉主体

## 4. 技术方案

### 技术栈

| 维度 | 选型 | 理由 |
|------|------|------|
| 框架 | **Next.js 14 (App Router)** | SSR/SSG、SEO 友好、团队熟悉（DevToolBox 同栈） |
| 样式 | **Tailwind CSS + Framer Motion** | Tailwind 快速开发 + Framer Motion 优雅动画 |
| 国际化 | **next-intl** | Next.js 生态最佳 i18n 方案，支持路由级切换 |
| RSVP 后端 | **Vercel Serverless + Vercel KV/Postgres** | 零运维，免费 tier 足够 |
| 图片 | **Next.js Image + Vercel Blob** | 自动优化、CDN 加速 |
| 部署 | **Vercel** | 自动部署，自定义域名，免费 SSL |
| 域名 | 可选购买（如 huayang-wedding.com）或用 Vercel 子域名 |

### 为什么自建而非用 Zola/WithJoy？

1. **完全定制** — 模板平台无法实现 100% 定制的视觉设计
2. **双语原生** — 平台的双语支持都不理想（需要手动复制内容）
3. **无品牌水印** — 免费平台通常有 "Powered by XXX"
4. **永久保存** — 平台可能关闭或收费，自建可永久保留
5. **团队有能力** — 已有 Next.js + Vercel 全套经验

## 5. UX 设计详细说明

### 首屏 (Hero)

```
┌─────────────────────────────────────┐
│           [全屏婚纱照]               │
│                                      │
│        Huayang  &  [Partner]         │
│          华阳 · [伴侣名]             │
│                                      │
│         2026 年 5 月 23 日            │
│         May 23, 2026                 │
│                                      │
│      ⏰ XX 天 XX 时 XX 分 XX 秒      │
│                                      │
│          ↓ 向下滚动                   │
│  [ZH/EN]                             │
└─────────────────────────────────────┘
```

### 我们的故事 (Our Story)

- 垂直时间线布局，左右交替
- 每个节点：日期 + 标题 + 一段话 + 一张照片
- 滚动到视口时 fade-in 动画
- 内容由 Human 提供（模板中用 placeholder）

### 婚礼信息 (Wedding Details)

- 卡片式布局（2-3 列）
- 每张卡片：图标 + 标题 + 详情
  - 📅 日期与时间
  - 📍 地点（含嵌入地图）
  - 👔 着装要求
  - 🅿️ 停车信息

### RSVP

- 简洁表单：姓名、人数、出席意向（出席/不出席）、饮食偏好（可选）、祝福语（可选）
- 提交后显示温馨确认信息
- 数据存储到 Vercel KV/Postgres，Human 可在管理后台查看

### 照片画廊

- 瀑布流 / Masonry 布局
- 点击放大 Lightbox
- 分类标签（婚纱照 / 日常 / 婚礼现场）
