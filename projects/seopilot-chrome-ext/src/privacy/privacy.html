Privacy Policy

SEOPilot Lite — SEO Analyzer | Last updated: March 2026

TL;DR: SEOPilot Lite does not collect, store, or transmit any user data. All analysis happens 100% in your browser.

1. Data Collection

SEOPilot Lite does not collect any personal data. Specifically, we do not collect:

  • Browsing history or URLs you visit
  • Page content or HTML
  • Personal information (name, email, etc.)
  • Cookies or tracking identifiers
  • Usage analytics or telemetry

2. How the Extension Works

When you click the SEOPilot Lite icon:

  • A content script analyzes the currently active tab's DOM (HTML structure)
  • The analysis runs entirely in your browser — no data is sent to any server
  • Results are displayed in the popup and discarded when you close it

3. Permissions Used

SEOPilot Lite requests the minimum permissions needed to function:

  • activeTab — Access the current tab only when you click the extension icon
  • scripting — Run the SEO analysis script on the current page
  • storage — Save your settings (check toggles) locally in Chrome

We do not request broad permissions like "Access all websites" or "Read browsing history."

4. Data Storage

The only data stored locally is your settings (which SEO checks are enabled/disabled). This data:

  • Is stored using Chrome's built-in chrome.storage.local API
  • Never leaves your device
  • Can be cleared by uninstalling the extension

5. Third-Party Services

SEOPilot Lite does not use any third-party services, APIs, analytics, or tracking tools.

6. Changes to This Policy

If we ever change this privacy policy, we will update the "Last updated" date above. Any changes that affect data collection will be clearly communicated through extension updates.

7. Contact

If you have questions about this privacy policy, contact us at support@seopilot.app.
